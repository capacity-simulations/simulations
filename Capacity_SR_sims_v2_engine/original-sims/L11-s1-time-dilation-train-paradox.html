Simulation 1 — Time Dilation, train paradox (guided-inquiry)
Concept
A light clock's round-trip time is shortest in its rest frame (proper time) and longer by γ in any frame where the clock moves, because the light traverses a longer diagonal path at the same speed c.
Serves
Derive time dilation (via Pythagorean geometry of the light path); explain frame dependence of measured time intervals; distinguish invariant proper time from coordinate-dependent round-trip time.
Inquiry
Observer A (train) and Observer B (platform) each time the same pulse with their own clock. Will they agree? If not, who measures the longer time — and why?
Misconception
"Moving clocks run slow — so just figure out which frame is moving." (No frame is uniquely the moving one; the proper time belongs to the frame where both events are co-located.)

A friend says…

▾

"In B's frame, A is moving, so A's clock runs slow. That means B's clock runs fast relative to A's."

Switch to View 1 (A's frame). According to A, who is moving? Whose clock runs slow now?

Both observers say the other's clock runs slow. Time dilation is symmetric — each frame sees the other's processes take longer. The phrase "moving clocks run slow" is frame-dependent, not absolute, so there's no contradiction.

"A's frame must be special — time runs slower there than everywhere else, which contradicts Einstein's first postulate."

What is your friend assuming about the two events being measured?

Your friend assumes one pair of events is being compared in both frames. A's clock records the shorter time not because A's frame is special, but because A's clock is present at both events (emission and return happen at the same place in A). Any frame's local clock would measure the proper time for events co-located in that frame. No frame is privileged — the first postulate holds.
