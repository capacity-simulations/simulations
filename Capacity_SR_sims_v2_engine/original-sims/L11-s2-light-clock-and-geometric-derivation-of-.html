Concept
The time-dilation factor γ emerges geometrically from applying the Pythagorean theorem to the diagonal light path inside a moving light clock.
Serves
Derive time dilation (via the equivalent light-clock argument); explain the frame dependence of measured time intervals; distinguish proper time (invariant, one clock present at two events) from coordinate-dependent time.
Inquiry
Rest-frame bounce → moving-frame diagonal → Pythagoras → t = γ t₀, built up in five lecturer-driven additive stages while the animation and v-slider stay live.
Misconception
"Moving clocks run slow — so to find which frame measures the proper time, just figure out which one is moving." Neither frame is uniquely "the moving one"; the proper time is recorded by the single clock present at both events.

Guided inquiry

▾

Play, drag v, and step the build stage at any time — nothing is locked.

Stage 1. In the clock's rest frame the pulse goes straight up and back: distance 2h at speed c, so t₀ = 2h/c. One clock is present at both events → this is the proper time.

Stage 2–3. In the lab frame the clock moves, so the same pulse takes a diagonal path. Light still travels at c — is the diagonal longer or shorter than the vertical 2h?

Stage 4. Read off the three sides of one half-trip: vertical = h, base = vt/2, hypotenuse = ct/2.

Stage 5. Apply Pythagoras to the right triangle and solve for t. What does it equal?

Discussion prompt

▾

"A friend says: 'Moving clocks run slow — so to find which frame measures the proper time, just figure out which one is moving.'"

From the simulation: is either frame uniquely "the moving one"?

Neither frame is uniquely "the moving one." Each frame sees the other's clock as moving — the first postulate guarantees no frame is special.

The proper time t₀ is measured by the clock that is present at both events (the same clock records the pulse leaving and returning). It's not about which frame "is moving" — it's about which single clock is at both events.

Symbols
  • h = mirror separation (frame-invariant)
  • c = light speed (invariant, same in every frame)
  • t₀ = 2h/c = rest-frame round-trip time — the proper time (one clock, two events)
  • t = lab-frame round-trip time (the moving clock, measured from outside)
  • v = clock speed in the lab frame (v > 0 → rightward)
  • γ = 1/√(1 − v²/c²) ≥ 1 for 0 ≤ v < c (frame-invariant — both frames agree on γ)
