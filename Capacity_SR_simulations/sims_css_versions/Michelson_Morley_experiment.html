About This Simulation

This interactive simulation demonstrates the Michelson–Morley experiment (1887), one of the most important experiments in physics history.

A beam of light is split into two perpendicular arms by a beam splitter (BS). Each beam reflects off a mirror (M₁, M₂) and recombines at a detector, producing an interference pattern.

In the 19th century, physicists believed light traveled through a medium called the “luminiferous aether.” If so, the Earth’s motion through this aether would create a “wind” that changes the effective speed of light differently in each arm, producing a measurable fringe shift when the apparatus is rotated.

Aether Prediction mode shows the expected fringe shift using Galilean velocity addition — this prediction is wrong.

Actual (Relativity) mode shows the real result: no fringe shift at any angle. The speed of light is the same in all directions, leading directly to Einstein’s Special Relativity (1905).

Note: β (v/c) is greatly exaggerated here for visibility. Earth’s actual orbital β ≈ 10⁻⁴.
