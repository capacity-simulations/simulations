About the simulation window

This simulation shows how distance stays the same when you translate or rotate a Cartesian coordinate system over a fixed pattern of stars.

Stars are fixed in the background. Drag the pink origin to move the axes, use the rotation slider (or auto-rotate) to turn them, and read each star’s coordinates in the panel as the frame changes.

d² = Δx² + Δy²

Green chords label the distance d between connected stars. That value does not change as you move or rotate the axes—the hallmark of a Euclidean invariant.

Rotations use an orthogonal matrix with RTR = I. This is the classical stepping stone toward Special Relativity, where the spacetime interval replaces ordinary distance.
