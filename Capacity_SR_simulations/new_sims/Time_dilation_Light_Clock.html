About This Simulation

This animation demonstrates time dilation — one of the most striking predictions of Einstein's Special Theory of Relativity — through the classic light clock thought experiment.

Two identical light clocks are compared in the platform (lab) frame: one at rest (top) and one carried by a train moving at speed v (bottom). A photon bounces between mirrors separated by height h.

Rest clock period: TA = 2h / c Moving clock period: TB = 2hγ / c = γ · TA

Because the speed of light c is identical in both clocks, the moving clock's photon must cover a longer diagonal path per tick — so each tick takes longer and the moving clock visibly runs slow. Both photons move at the same on-screen speed; only the path length (and therefore the period) differs.

Adjust β = v/c and mirror height h using the controls on the right panel.
