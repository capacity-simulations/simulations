About the simulation
This interactive explores 2D linear algebra in ℝ². Define vectors a and b, then form linear combinations v = c₁a + c₂b. The view shows inner product, angle, projection onto b, and linear independence (determinant). Drag vector tips on the canvas or edit values in the sidebar; use the sliders to move v through the plane spanned by {a, b}.
