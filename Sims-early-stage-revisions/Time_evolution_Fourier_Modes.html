⚛️ About the Simulation

Physics

This simulates a quantum particle confined in a 1D infinite potential well (particle in a box) of width L=1. The wave function ψ(x,t) is expanded in energy eigenstates φₙ(x) = √2 sin(nπx) with energies Eₙ = n²π². Time evolution applies phase factors e−iEₙt to each mode. The system uses natural units (ℏ = 2m = L = 1) and exhibits quantum revivals at Trev = 2/π ≈ 0.637.

Visuals
  • Purple filled curve — probability density |ψ(x,t)|²
  • Blue line — real part Re[ψ(x,t)]
  • Green dashed line — expectation value ⟨x⟩
  • Red dashed lines — infinite potential walls at x=0 and x=1
  • Bar chart below — mode coefficients |aₙ|² with rotating phasor arrows showing each mode's time-dependent phase
Controls
  • Initial State — choose from linear ramp, Gaussian, left-half step, or ground state
  • Nmax slider — number of Fourier modes (1–40)
  • Time slider — scrub through the time evolution
  • Speed selector — playback speed (0.25× to 5×)
  • Play/Pause — toggle animation (or press Space)
  • Reset — return to t = 0
