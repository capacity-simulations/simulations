About this simulation

This simulation models a quantum particle in a one-dimensional infinite square well, where the walls are impenetrable and only discrete stationary states are allowed.

Physics
  • Energy is quantized: En = n²E1.
  • The wavefunction is ψn(x, t), and measurement probability is |ψ|².
  • In physical mode, oscillation scales as ωn = n²ω0.
Visuals
  • Top plot: wave function ψ(x, t) inside the box walls.
  • Bottom plot: probability density |ψ|² with optional measurement histogram.
  • Left axis: discrete energy levels, with the selected level highlighted.
Controls
  • Choose energy level n and adjust measurement rate.
  • Toggle override mode to set animation ω directly with the speed slider.
  • Show/clear histogram, then use Pause/Reset to control the run.

The display links wave evolution to measured outcomes, showing how repeated position measurements build the probability distribution predicted by |ψ|².
