About the simulation

Physics. This uses the lecture convention for the finite well: V(x) = -V0 for |x| < a and V(x) = 0 outside, so the well has width 2a. Bound states have negative energy, oscillate inside the well, and decay exponentially in the classically forbidden region |x| > a.

Display. The upper panel shows the potential and the selected bound-state energy. The lower panel shows the actual wavefunction and probability density with their own labeled vertical scales, so no hidden peak-rescaling is used.

Assumptions. Numerical values are shown using a fixed particle mass equal to one electron mass. Keeping the mass fixed narrows the model compared with the symbolic lecture notes, but makes the plotted energies and decay lengths concrete.

Controls. V0 changes the well depth and 2a changes the well width. Animation speed only changes the phase rotation of Ψ. Choose n with ◀ ▶, the energy list, or ↑/↓. Reset restores defaults; Pause or space stops time.
